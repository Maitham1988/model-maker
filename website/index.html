Capabilities

Conversational Interface

Natural language conversation with streaming responses. Users describe symptoms and receive structured guidance.

Knowledge Retrieval

RAG system with ONNX embeddings searches verified medical documents before generating responses. Reduces hallucination.

Persistent Memory

Stores patient-reported allergies, conditions, and medications. Provides personalized guidance across sessions.

Readability

Dark interface designed for low-light and field conditions. High contrast. Large touch targets. Screen-reader compatible.

Any Device

Responsive interface works on phones, tablets, laptops, and desktops. No installation beyond Python.

Real-Time Streaming

Server-Sent Events deliver token-by-token responses. Users see output immediately without waiting for full generation.
