 Install on Mac

Open Terminal and paste:

curl -fsSL https://raw.githubusercontent.com/Maitham1988/model-maker/main/scripts/install-mac.sh | bash

Installs the app + AI model (~5 GB). Creates a Desktop launcher. Done.

Terminal: press ⌘+Space, type Terminal, press Enter
