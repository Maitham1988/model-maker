Privacy Policy

Effective: March 2026 · Last updated: March 2026

1. Overview

Model Maker is designed to operate entirely offline on the user's device. This privacy policy explains what data is and is not collected.

2. Data We Do Not Collect

Model Maker does not collect, transmit, or store any of the following:

  • Conversations or chat messages
  • Personal information (name, email, phone, location)
  • Medical or health information
  • Device identifiers or hardware information (sent externally)
  • Usage analytics or telemetry
  • Cookies or tracking data
  • IP addresses

3. Local Data Storage

Model Maker stores the following data locally on your device only:

  • Conversations: Stored in a local SQLite database on your device. Never transmitted.
  • Memory facts: User-provided facts (e.g., allergies, conditions) stored locally for personalized responses.
  • Configuration: Application settings stored in a local JSON file.

All local data can be deleted at any time by removing the application data folder.

4. Network Activity

After initial setup, Model Maker requires no internet connection. The only network activity occurs during:

  • Initial download: Downloading the application and AI model from our servers or GitHub.
  • Optional updates: Checking for updates if the user chooses to do so.

No data is sent from your device to any server during normal operation.

5. Downloads

Model Maker is free and open source. All downloads are available at no cost from our GitHub repository. No payment processing is involved.

6. License Verification

The optional license system generates a hardware identifier locally using a one-way hash (SHA-256) of device characteristics. This identifier never leaves your device unless you contact support.

7. Open Source Transparency

Model Maker is open source under the Apache 2.0 license. The complete source code is available for audit at github.com/Maitham1988/model-maker. Anyone can verify these privacy claims by reviewing the code.

8. Children's Privacy

Model Maker does not knowingly collect any data from children or any other users. No data is collected from any user of any age.

9. Changes to This Policy

If this policy changes, the updated version will be posted on this page with a new effective date. Because Model Maker collects no data, meaningful changes are unlikely.
